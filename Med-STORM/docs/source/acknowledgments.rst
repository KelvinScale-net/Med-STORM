Acknowledgments
===============

Med-STORM is built upon the work of many researchers and open-source projects. We would like to acknowledge and thank the following open-source projects that are directly integrated into our codebase:

Core Inspirations
----------------

STORM (Synthesis of Topic Outlines through Retrieval and Multi-perspective Question Asking)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- **Source**: `GitHub Repository <https://github.com/stanford-oval/storm>`_
- **License**: MIT License
- **Contribution**: Inspired the core architecture and evidence synthesis approach.

Med-PRM (Medical Path Retrieval and Mining)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- **Source**: `GitHub Repository <https://github.com/eth-medical-ai-lab/Med-PRM>`_
- **License**: MIT License
- **Contribution**: Influenced our approach to medical evidence retrieval and processing.

Key Open Source Components
-------------------------

PubMed E-utilities
~~~~~~~~~~~~~~~~~~
- **Source**: `NCBI E-utilities <https://www.ncbi.nlm.nih.gov/books/NBK25501/>`_
- **License**: Public Domain
- **Role**: Provides programmatic access to biomedical literature from PubMed.

Open Source Community
--------------------

We are grateful to the broader open-source community for their contributions to the tools and libraries that make Med-STORM possible, including:

- **Python** and its ecosystem (MIT/PSF License)

- **pytest** for testing (MIT License)

------------------

1. **STORM**: 
   ```

     title={STORM: Synthesis of Topic Outlines through Retrieval and Multi-perspective Question Answering},
     author={Author, A. and Author, B.},
     booktitle={Proceedings of...},
     year={2023}
   }
   ```

If you use Med-STORM in your research, please cite our work:

```

  author = {Your Name and Contributors},
```
