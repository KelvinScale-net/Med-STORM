Key Concepts
-----------

Med-STORM is built around several key concepts that are important to understand:

1. **Evidence Tiers**
   - **Tier 1**: Maximum confidence (PubMed, ClinicalTrials.gov)
   - **Tier 2**: High confidence (Guidelines, trusted organizations)
   - **Tier 3**: Supplementary sources (Conference proceedings, expert opinions)

2. **Search Workflow**
   - Query formulation
   - Evidence collection
   - Evidence processing
   - Report generation

3. **Confidence Scoring**
   - Source reliability
   - Study design
   - Sample size
   - Publication date
   - Risk of bias
